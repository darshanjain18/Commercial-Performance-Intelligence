# 📊 Commercial Performance Intelligence Dashboard
> ⚡ Power BI | 🛒 E-commerce Analytics | 📅 2022 Sales Data

## 🔍 Overview
An end-to-end commercial analytics dashboard built in Power BI, analyzing **113,000+ e-commerce transactions** across Office Supplies, Furniture, and Technology categories. Tracks sales performance, delivery efficiency, and profitability across 4 US regions.

## 💡 Key Insights
- 💰 Total Revenue: **$23.2M** | Total Profit: **$2.6M**
- 👥 Segments analyzed: Consumer, Corporate, Home Office
- 🚚 Delivery performance tracked across 4 status types
- 🗺️ Regional breakdown: East, Central, South, West

## ✨ Features
- 📌 KPI cards with YoY performance tracking
- 🌎 Regional sales heatmap with lat/long geo data
- 📦 Delivery status analysis & late shipment trends
- 📈 Category & segment profitability breakdowns
- 🎛️ Dynamic slicers for region, segment, and category

## 🗃️ Dataset
| Field | Description |
|-------|-------------|
| 📋 113,270 rows | Full year 2022 order data |
| 🔢 21 columns | Customer, product, shipping, financials |

## 🛠️ Tools Used
- 🟡 Power BI Desktop
- 📐 DAX for calculated measures
- 🔄 Power Query for data transformation

## 🚀 How to Use
1. Clone this repo  
2. Open `report/Commercial_Performance_Intelligence.pbix` in Power BI Desktop  
3. Refresh data source pointing to `/data/ecommerce_data.csv`
